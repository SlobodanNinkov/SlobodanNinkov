Recognition Memory Task

Tests explicit long-term memory using a study-test paradigm.

  • Study phase: 15 words appear one at a time — memorise them.
  • A short distractor period follows.
  • Test phase: 30 words appear (15 old + 15 new).
  • Click OLD if you saw it in the study phase, NEW if not.
  • Accuracy and signal-detection sensitivity (d′) are computed.
