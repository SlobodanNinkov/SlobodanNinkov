Serial Reversal Tasks

Four sub-tasks that challenge you to process familiar sequences backwards.

  • Word Reversal: Spell each word backwards.
  • Days Backward: Type the days of the week in reverse order.
  • Count –3: Starting from 30, enter each value as you count back by 3.
  • Count –7: Starting from 100, count back by 7 for 12 steps.
